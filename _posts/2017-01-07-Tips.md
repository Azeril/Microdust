这篇博文主要是为了记录我写代码时遇上的一些小经验，怕忘了，就在这记下来吧。

#### 1、我想自己写一个弹框，然后相对浏览器窗口静止，只需要在css样式里写入如下代码：

```
body{ position: relative; }
div{ position: fixed; }
```
